Associação com a Lista de Materiais

Após sua criação, a Sub-lista de Fornecimento deve ser relacionada com a Lista de Materiais.

Exemplo

Adotemos uma Lista de Materiais qualquer, recém-criada (nenhuma Sub-lista de Fornecimento)

Fornecedor

Se uma nova Sub-lista de Fornecimento para o Fornecedor 'Petrobras' for criada, o conjunto de Sub-listas de Fornecimento relacionadas à Lista de Material deve ser:

Fornecedor
Petrobras

Se uma outra Sub-lista de Fornecimento, desta vez para o Fornecedor 'XYZ' for criada, o conjunto de Sub-listas de Fornecimento relacionadas à Lista de Material deve ser:

Fornecedor
Petrobras
XYZ
